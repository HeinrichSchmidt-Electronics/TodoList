# Task Manager (консольный список задач)

Консольное приложение на C++ для управления списком задач. Задачи сохраняются в файл `config.json` и загружаются автоматически при запуске. Навигация по меню реализована через паттерн State.

## Инструкция по запуску

1. Убедиться, что установлена библиотека `nlohmann/json` (через vcpkg, Conan или локально в `include/`).
2. Собрать проект (пример для g++):
   ```bash
   g++ -std=c++17 main.cpp -o task_manager
   ```
   Либо открыть проект в Visual Studio / CLion и собрать через IDE.
3. Запустить исполняемый файл:
   ```bash
   ./task_manager
   ```
   На Windows:
   ```bash
   task_manager.exe
   ```
4. При первом запуске автоматически создастся файл `config.json` в той же папке.

## Реализованные функции

- Главное меню: переход к списку задач или выход из программы
- Просмотр списка задач с отметкой о выполнении
- Добавление новой задачи
- Отметка задачи как выполненной
- Удаление задачи по номеру
- Автоматическое сохранение и загрузка данных из `config.json`

## Пример работы

```
-- Главное меню --
1. Открыть заметки
2. Выйти
Выберите действие: 1

-- Задачи --
[Список пуст]

1. Добавить задачу
2. Отметить выполненной
3. Удалить задачу
4. Вернуться назад
Выберите действие: 1
Введите задачу: Сделать лабораторную по ООП

-- Задачи --
1. [ ] Сделать лабораторную по ООП

1. Добавить задачу
2. Отметить выполненной
3. Удалить задачу
4. Вернуться назад
Выберите действие: 2
Введите номер задачи: 1

-- Задачи --
1. [x] Сделать лабораторную по ООП
```
