\# TodoList (Консольный планировщик задач)



Простой и эффективный консольный планировщик задач на C++, использующий паттерн проектирования \*\*State (Состояние)\*\* для гибкого управления экранами меню и библиотеку \*\*nlohmann/json\*\* для постоянного хранения данных.



\## 🚀 Особенности

\- \*\*Паттерн State\*\*: архитектурно правильное разделение логики интерфейса. Переходы между главным меню и списком задач изолированы в классах состояний.

\- \*\*Сохранение данных\*\*: автоматическое создание, чтение и перезапись файла `config.json` при любых изменениях в списке задач.

\- \*\*Управление задачами\*\*: добавление новых пунктов, удаление, а также отметка задач как выполненных `\[x]`.

\- \*\*Поддержка UTF-8\*\*: корректное отображение кириллицы в консоли Windows.



\## 🛠️ Технологии и требования

\- \*\*Язык\*\*: C++20

\- \*\*Среда разработки\*\*: Visual Studio 2022 (используется современный формат решений `.slnx`)

\- \*\*Зависимости\*\*: `nlohmann.json` (пакет NuGet)

\- \*\*Операционная система\*\*: Windows
